# Política de Segurança

## Versões Suportadas

Atualmente, apenas a versão mais recente da branch `main` recebe atualizações de segurança.

| Versão | Suportada          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reportando uma Vulnerabilidade

Se você encontrou uma falha de segurança, por favor NÃO abra uma Issue pública.

Envie um e-mail para cbucazio@gmail.com com detalhes do problema. Nós responderemos em até 48 horas.
